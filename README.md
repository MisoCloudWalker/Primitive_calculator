## Python Calculator (пет-проект)
Простой, но мощный консольный калькулятор на Python с поддержкой основных математических операций и логгированием вычислений.

## Особенности
- 6 базовых операций: сложение, вычитание, умножение, деление, возведение в степень, остаток от деления
- Автоматическое логгирование: все вычисления сохраняются в файл
- Обработка ошибок: защита от деления на ноль и неверного ввода
- Гибкий ввод: поддержка разных форматов ввода степени (** и ^)
- История вычислений: полный журнал всех операций с временными метками
- Простой интерфейс: интуитивно понятная консольная программа

## Установка
1. Сохраните код в файл `calculator.py`
2. Запустите: `python calculator.py`

## Доступные операции
| Операция | Описание | Пример |
|----------|----------|--------|
| `+` | Сложение | `5 + 3 = 8` |
| `-` | Вычитание | `10 - 4 = 6` |
| `*` | Умножение | `7 * 6 = 42` |
| `/` | Деление | `15 / 3 = 5` |
| `**` | Возведение в степень | `2 ** 3 = 8` |
| `^` | Возведение в степень (альтернатива) | `2 ^ 3 = 8` |
| `%` | Остаток от деления | `10 % 3 = 1` |

## Логирование
Все операции автоматически записываются в файл `all operations.txt"`
### Пример
```
5 + 3 = 8 at 2024-01-15 14:30:25
10 / 2 = 5 at 2024-01-15 14:31:10
2 ** 3 = 8 at 2024-01-15 14:32:45
```
## Архитектура
Калькулятор использует:
- Словарь операций для динамического вызова функций
- Обработку исключений для устойчивости к ошибкам
- Контекстный менеджер для безопасной работы с файлами
- Форматирование вывода для лучшей читаемости
